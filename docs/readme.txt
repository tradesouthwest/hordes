<h2>Admin Tips</h2>
<dl>
 	<dt><b>User Levels</b></dt>
 	<dd><strong><span style="color: #000000;"><span class="info">Upon install of Hordes you may want to set the option to redirect upon login so that you can go directly to the form page.</span><sup>*</sup></span></strong></dd>
 	<dd></dd>
 	<dt><b>Setup</b></dt>
 	<dd>This section describes how to install the plugin and get it working.</dd>
 	<dd>1. The shortcode to add inside of your post or page to show list is: [hordes_list] </dd>
 	<dd>2. The shortcode to add inside of your post or page to show a custom alphabetic search bar is: [hordes_search_bar] </dd>
 	<dd>3. To add a front end form to a page use shortcode: [hordes_submit_post] </dd>
 	<dd>4. Go through all options in Hordes Settings to activate what you want to use.</dd>
 	<dd></dd>
 	<dt><b>Check to not show favicons in the Tall list (Optional)</b></dt>
 	<dd>By removing favicons in your list it saves some bandwidth. Favicons are loaded from the Google favicons generator. More info at: https://www.labnol.org/internet/get-favicon-image-of-websites-with-google/4404/</dd>
 	<dt><b> Check to Show a lower profile list of links.</b></dt>
 	<dd>Shorter list displays only the title and the link. .</dd>
 	<dt><b>Color for List Title</b></dt>
 	<dd>If field is left unchanged then the titles will stay the same as the theme default titles color.</dd>
 	<dt><b>Color for List Links</b></dt>
 	<dd>Colors can be set to help brand your page to match your theme.</dd>
 	<dt><b>What Page is shortcode 'hordes_list' on?</b></dt>
 	<dd>This must be set if you want a link on the form page. The link will show on the popup "Success" message after submitting your new link.</dd>
 	<dt><b>What Page is shortcode 'hordes_submit_post' on?</b></dt>
 	<dd>Set page to be redirected upon login. Turn on or off Using the next setting below.</a></dd>
 	<dd><small>This is only a convenience to administrators to save time when you are in a hurry to add links.</small></dd>
 	<dt><b>Redirect on Login</b></dt>
 	<dd>Check to Redirect directly to Form Submit Page. Only works for logged In Administrators. </dd>
 	<dt><b>Front Side Submit Form</b></dt>
 	<dd>To add a front end form to a page use shortcode: [hordes_submit_post] <br>
The form will allow you to add title, link and tags to your post, but, only Administrators and Editors will see the form on the public side of your Website. A Featured Image maybe added to every listing but you will need to do this from the editor so that you can control the assets for each picture.</dd>
 	<dt><b>Show Alpha Checkbox Search Bar</b></dt>
 	<dd>The shortcode to add inside of your post or page to show a custom alphabetic search bar is: [hordes_search_bar] <br>Works for categories as well as links and titles.</dd>
</dl>
